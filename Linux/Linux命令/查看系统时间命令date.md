语法：`date 【-d】【+格式化字符串】`
-d：按照指定的字符串给定信息，一般用于日期计算

| 格式化字符串 | 含义 |           
| ------ | --------------- |
| %Y     | 年                   |
| %y     | 年份后两位数字|        
| %m     | 月份              |     
| %d     | 日               |     
| %H     | 小时              |     
| %M     | 分钟              |     
| %S     | 秒               |     
| %s     | 自1970-1-1到现在的秒数 |    

支持的时间标记：
* year 年
* month 月
* day 天
* hour 小时
* minute 分钟
* second 秒
