# 前言
有些时候，我们需要将本地仓库直接关联为新的远程仓库，这时候就需要使用相关的Git命令。

# 从本地添加远程仓库
## `git remote add <shortname> <url>`
将本地仓库上传并添加为远程仓库（**前提是已创建远程仓库，并获得链接**）
![[Pasted image 20251202145037.png]]

## `git remote -v`
查看当前仓库对应的远程仓库的别名和地址
![[Pasted image 20251202145056.png]]

## `git branch -M <branch>`
指定该仓库的分支名称
![[Pasted image 20251202145119.png]]

## `git push -u <shortname> <本地仓库分支>:<远程仓库分支> `
将本地仓库里面的内容推送到远程仓库内
>[!note]
>若两个分支名相同，可合并为一个分支表示

![[Pasted image 20251202145501.png]]
![[Pasted image 20251202145640.png]]
