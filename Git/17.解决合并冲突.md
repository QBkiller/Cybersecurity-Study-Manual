# 前言
在Git的多分支开发中，总会出现两个分支对同一个文件进行修改，这样会在分支合并时产生冲突：
![[Pasted image 20251206103425.png]]
>[!note]
>这是Git不知道该如何合并时出现的错误
>需要我们手动合并
