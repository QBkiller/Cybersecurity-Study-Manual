# 确定你的Linux系统
在Linux系统下的Git安装并不需要访问官网下载安装包，基本上各个Linux系统的包管理器软件库里面默认包含了Git，只需要使用命令行安装即可。
>[!warning]
>不同的Linux系统的包管理器**并不相同！**
>Ubuntu/Kali/其他Debian：apt
>RHEL/CentOS/其他Red Hat：dnf或yum
>Arch：pacman

# 安装
我们是网安人员，最为推荐的Linux系统当然是Kali，但是由于Kali自带了Git，所以接下来的教程以Ubuntu为主。

---

输入命令：
```shell
sudo apt update #sudo用来提高权限(使用时要输入当前用户密码)，此命令用于更新包管理器
```

![[Pasted image 20251121104722.png]]

---

输入命令：
```shell
sudo apt install -y git #安装Git,-y表示确认，若不使用-y，需手动确认
```
![[Pasted image 20251121105143.png]]

---

输入命令：
```shell
git -v #此命令查看Git的版本，也是为了确认Git是否成功安装
```
![[Pasted image 20251121105320.png]]
