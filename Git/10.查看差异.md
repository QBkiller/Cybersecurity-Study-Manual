# git diff
默认查看**工作区**和**暂存区**之间的差异

# git diff HEAD
比较**工作区**和**仓库**之间的差异

# git diff --cached/git diff --staged
比较**暂存区**和**仓库**之间的差异

# git diff <commit_hash> <commit_hash>
比较两个**仓库版本**之间的差异
>[!note]
>实际开发过程中并不会查找版本哈希值并比较
>而是使用：git diff HEAD~n HEAD
>HEAD^表示上一个版本
>HEAD~n表示前n个版本

# git diff <branch_name> <branch_name>
比较不同**分支**之间的差异

>[!note]
>git diff HEAD和git diff --cached两个命令之间有着本质的区别
>前者对比：工作区（当前本地修改**但未提交**暂存区文件）vs HEAD（**指向的最新提交版本**）
>后者对比：暂存区（当前本地修改**且已提交**暂存区文件）vs HEAD（**指向的最新提交版本**）
