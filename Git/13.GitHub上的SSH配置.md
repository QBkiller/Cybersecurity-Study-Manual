# 配置过程
## 进入家目录
![[Pasted image 20251201192932.png]]
当MINGW64后面的字符变为~，说明已经进入家目录

>Windows系统下的家目录为：C:\Users\QBhunter（用户名）

## 进入并生成SSH密钥
![[Pasted image 20251201193808.png]]
>[!note]
>ssh-keygen是一个专门用于生成、管理和转换SSH认证密钥的工具。
>-t 指定RSA类型的SSH认证密钥对
>-b 指定4096位长度

![[Pasted image 20251201194204.png]]
生成密钥中……
此处可以输入保存密钥的文件名，可以直接enter键选择默认（==前提是你的电脑里没有默认的密钥对，否则会覆盖！==）、

![[Pasted image 20251201194427.png]]
可以输入自己的密码，重复输入后密钥对就生成完毕。

## 公钥上传GitHub
![[Pasted image 20251201194809.png]]
![[Pasted image 20251201194725.png]]
进入.ssh文件夹，会发现生成的两个文件，分别是私钥和公钥。
>[!warning]
>私钥文件**一定不要给任何人！！！**

随后就可以将自己的公钥文件打开复制，并上传至GitHub。

![[Pasted image 20251201195723.png]]
